# Pixlel AI 🧠

**Умный чат-бот с собственным "мозгом"** — думает шаг за шагом, использует инструменты, помнит историю и рассуждает как человек.

## Особенности
- Chain of Thought + ReAct мышление
- Долгосрочная память
- Инструменты (поиск, калькулятор, код и др.)
- Локальный запуск через Ollama
- Красивый Gradio интерфейс

## Быстрый старт

```bash
pip install -r requirements.txt
cp .env.example .env

# Запуск
python main.py
```

## Структура проекта
```
src/
├── brain/      # Главный "мозг"
├── models/     # Работа с LLM
├── ui/         # Интерфейсы
└── utils/      # Промпты и утилиты
```
